## [eventEmitterTest.js](eventEmitterTest.js)

Node.js 事件循环-事件驱动程序实例。

疑惑在于该程序是输出是：

```
$ node main.js
连接成功。
数据接收成功。
程序执行完毕。
```

既然 Node.js 的每一个 API 都是异步的，那么为何“程序执行完毕。”是最后输出而不是最先输出？
